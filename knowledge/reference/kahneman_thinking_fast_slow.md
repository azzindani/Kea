# Standards & Authorities
*   **Source**: Daniel Kahneman (Nobel Laureate).
*   **Focus**: Cognitive biases, heuristics, and the "Human Error" in economic decision-making.

## Core Framework
1.  **System 1 (Fast)**: Automatic, emotional, and relies on pattern matching. Excellent for survival; dangerous for complex logic.
2.  **System 2 (Slow)**: Effortful, logical, and computational. Prone to laziness.
3.  **WYSIATI (What You See Is All There Is)**: The tendency to ignore "Absent" evidence and build a coherent story based only on visible data.
4.  **Prospect Theory**: The asymmetry between how humans perceive gains and losses (Loss Aversion).
5.  **Base-Rate Neglect**: Ignoring general probabilities in favor of specific, vivid details.

## Application in Kea
*   **System 2 Engagement**: When a task is "Critical," the agent must explicitly trigger "Slow Thinking" (DAG-based multi-step verification) to override "Fast" LLM patterns.
*   **Base-Rate Audit**: Before making a prediction, the agent must find the "Global Frequency" of the event.
*   **The Loss-Aversion Filter**: When recommending a "Stop" to a project, the agent must check if it is being too cautious due to irrational fear of loss.
*   **Evidence Check**: Explicitly search for "What is NOT here" before finalizing a report.
