# Standards & Authorities
*   **Key Figure**: Eli Whitney (Interchangeable Parts) / Henry Ford (The Assembly Line).
*   **Core Event**: The shift from "One-off" crafts to systemic production in late 18th century Britain.
*   **Focus**: Repeatability, precision, and the decoupling of the "Creator" from the "Product."

## Core Framework
1.  **Interchangeable Parts**: The realization that if every component (bolt, gear, logic module) is identical, the system can be repaired or scaled infinitely.
2.  **Specialization of Labor**: Breaking a complex task (e.g., making a pin) into 18 distinct, simple operations to increase speed and reduce error.
3.  **The Assembly Line**: Moving the "Work" through the "Workers" rather than the workers through the work.
4.  **Standardization of Units**: The creation of universal measurement systems to ensure components from different factories work together.

## Application in Kea
*   **The "Silicon Factory"**: Treat every agentic task as an assembly line. Input -> Triage -> Research -> Synthesis -> Output.
*   **Standardized Tools**: Just as interchangeable bolts revolutionized machines, standardized MCP tools revolutionize agents. If the interface is the same, the agent is interchangeable.
*   **The Artisanal Trap**: Avoid "Special-case" logic. If a problem occurs twice, create a "Standard Part" (a procedure or rule) to handle it for the entire swarm.
